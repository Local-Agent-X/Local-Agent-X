# Local Agent X

A self-hosted agent platform you run on your own machine. Supports multiple LLM providers. Includes voice (push-to-talk and full-duplex). Runs scheduled missions on cron. Builds and serves apps from a local workspace.

## Install

- Windows (cmd): `install.bat`
- Windows (PowerShell): `install.ps1`
- macOS / Linux: `install.sh`
