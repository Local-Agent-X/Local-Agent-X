Enable Sync
Automatically sync agent data to GitHub
Setup Guide
Step 1: Go to github.com/new and create a new private repository (e.g. "agent-memory"). Leave it empty — no README.
Step 2: Copy the repo URL and paste it below.
Step 3: Go to GitHub Settings → Fine-grained tokens and create a token scoped to only your memory repo with Contents: Read and write. This limits exposure if the token ever leaks.
Tip: If you also want your agent to work on other repos when asked, save a separate classic token as GITHUB_TOKEN in the Secrets tab.
Step 4: Paste the token below, enable sync, and click Save.
