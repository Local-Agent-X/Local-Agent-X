Browser Session
Where the agent's browser runs. Isolated = safest. Attach = convenient but larger blast radius.
Mode
Isolated — dedicated agent profile at ~/.sax/chrome-profile. Agent logs in separately. Page injection can only see what the agent itself logged into.
Attach mode is active. The agent runs against your real Chrome profile, so it inherits every login you have (Gmail, banks, internal dashboards). A compromised page or prompt injection can now act with your full identity.

Requires your regular Chrome to be closed — Chrome refuses two instances on one profile. Quit Chrome before any task that opens the browser, or switch back to Isolated.
